# TIMECALC 🕒

Calculadora de desfases horaria diseñada específicamente para el análisis de grabaciones de video (DVR) y sincronización con hora oficial.

## ✨ Características

- **Cálculo Automático:** Determina el desfase comparando la hora de un DVR con la hora oficial.
- **Conversión de Horas:** Encuentra fácilmente la hora de búsqueda en el DVR a partir de la hora real o viceversa.
- **Desfase Manual:** Permite ingresar desfases conocidos (años, meses, días, horas, etc.) sin necesidad de datos iniciales.
- **PWA Ready:** Instalable en dispositivos móviles y funcional sin conexión a internet.
- **Interfaz Moderna:** Diseño oscuro (Dark Mode) optimizado, responsivo y profesional.

## 🛠️ Tecnologías

- **Frontend:** HTML5, CSS3 (Custom Properties), Bootstrap 5.
- **Lógica:** JavaScript (jQuery), Moment.js (para cálculos precisos de tiempo).
- **Offline:** Service Workers y Manifest JSON.

## 🚀 Instalación y Uso

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en cualquier navegador moderno.
3. Para instalarlo como aplicación, utiliza la opción "Instalar aplicación" o "Añadir a la pantalla de inicio" en tu navegador móvil o escritorio.
